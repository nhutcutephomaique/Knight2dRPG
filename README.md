# Knight2DRPG

Unity 2D RPG project.

## Mở dự án

1. Mở Unity Hub.
2. Chọn `Add` và duyệt tới thư mục `t:\Game\Knight2DRPG`.
3. Mở dự án bằng Unity phiên bản tương thích (xem `ProjectSettings/ProjectVersion.txt`).

## Cấu trúc chính

- `Assets/` chứa tài nguyên và mã nguồn Unity.
- `Packages/` chứa dependencies Package Manager.
- `ProjectSettings/` chứa cài đặt dự án.

## Ghi chú

- Sử dụng `Unity Input System` nếu dự án yêu cầu điều khiển mới.
- Tránh chỉnh sửa file tự động sinh như `Library/`, `Temp/` và `UserSettings/`.

## Git

- Commit chỉ các file cần thiết trong `Assets/`, `Packages/`, `ProjectSettings/`.
- Không commit các thư mục và file được sinh tự động.
